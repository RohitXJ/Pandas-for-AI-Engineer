# 🐼 Pandas for AI Engineer: The Gateway to LLMs

Welcome to my learning journey! This repository is a structured pathway designed to take you from a data beginner to an AI-ready engineer, specifically focusing on the skills needed for Machine Learning and Large Language Models (LLMs).

## 🚀 The 6-Phase Journey

1.  **Phase 01: Foundations & Modern I/O** — Understanding how to load data and inspect it like a pro.
2.  **Phase 02: Data Cleaning for ML** — Fixing missing values and dirty data to prevent "Garbage In, Garbage Out."
3.  **Phase 03: Feature Engineering & EDA Strategy** — Creating signals and discovering relationships for AI models.
4.  **Phase 04: NLP & Text Data Prep** — The bridge to LLMs. Cleaning text, chat logs, and reviews.
5.  **Phase 05: Performance & Vectorization** — Scaling your code to handle millions of rows efficiently.
6.  **Phase 06: Production-Ready Formats** — Mastering Parquet, Arrow, and memory optimization.

## 📁 Repository Structure

*   `01-06/`: Each folder contains notebooks covering real-world scenarios.
*   `_linkedin_hub/`: A workspace for generating and tracking my progress on LinkedIn.

## 🤝 How to use this
Each notebook follows a **Scenario-Based** approach. I don't just write code; I solve an AI-related problem. If you are a future AI Engineer, follow the folders in order!
